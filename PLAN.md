# Baby Skin Lab — 컨셉 기획서 v1.0

> 작성: 2026-06-21 · 도메인: babyskinlab.com · 운영: 주식회사 빌리지베이비 (투명 공개)

## 1. 한 줄 정의

**"우리 아기 피부, 지금 어떤 상태? 뭘 해야 해?"에 60초 안에 답하는 판정·진단 도구 랩.**

글을 읽게 하는 사이트가 아니라, 부모가 몇 번의 탭으로 답을 얻는 사이트. 인터랙티브 도구가 주인공이고 글은 도구의 근거를 설명하는 보조 자산이다.

## 2. 포지셔닝 (3사 구도)

ruuve-guide와 포맷·키워드 타겟·콘텐츠가 구조적으로 달라 doorway 판정 리스크가 낮다. 운영사는 footer·About에 투명 공개한다.

## 3. 핵심 도구 5종 스펙

### 도구 1 — 아기 피부 증상 진단기 ⭐ 파일럿

- **URL**: `/tools/skin-symptom-checker/` (영문 slug — 신규 사이트는 글로벌 여지 고려, 한국어 UI)
- **입력 (4스텝 탭 선택)**:
  1. 부위 — 뺨/이마 · 목 주름/겨드랑이 · 입 주위/턱 · 기저귀 부위 · 팔꿈치/무릎 안쪽 · 두피 · 전신
  2. 모양 — 매끈한 빨강 · 좁쌀 융기(까끌) · 진물/딱지 · 갈라짐/거침 · 물집
  3. 시기·환경 — 월령(0-1/1-3/3-6/6-12/12+개월) + 계절/더운 환경 여부 + 시작 후 경과(1주 미만/1-4주/4주+)
  4. 아기 반응 — 가려워하며 비빔 · 평소와 같음 · 수면 방해
- **판정 로직 (rule-based 스코어링)**: 태열·땀띠·아토피·침독·기저귀발진·건조 6개 후보에 가중치 매트릭스로 점수 부여 → 상위 1-2개 "가능성 높음/중간" 표시. 확정 진단 아님을 명시.
  - 예: 뺨+매끈빨강+1-3개월+가려움없음 → 태열 高 / 목주름+좁쌀+여름 → 땀띠 高 / 4주+지속+가려움+수면방해 → 아토피 의심 → 진료 권고 강조
  - 로직 소스: ruuve-guide 감별 글들의 4단서 체계(부위·모양·계절·가려움)를 **표현은 새로 써서** 도구화 (텍스트 복제 금지)
- **출력 (결과 카드)**:
  1. 판정 — "태열 가능성이 높아요 (85점)" + 신뢰도 게이지
  2. 지금 할 일 3가지 — 액션 카드 (환경 온도·세정·보습 방향)
  3. 🚨 진료 신호 체크리스트 — 해당 시 즉시 진료 권고 (YMYL 안전장치)
  4. 케어 기준 — "이런 보습제를 찾으세요" 성분·스펙 기준 제시 → **기준 충족 예시 1개로 러베 제품** (예: "세라마이드 NP 2,000ppm 이상 예: 러베 5중 세라마이드 로션")
  5. 근거 링크 — 지원 글 2-3편
- **결과 공유**: URL 파라미터로 결과 재현 가능 (`?r=cheek-smooth-1m-noitch`) → 맘카페 공유 유도 = 백링크

### 도구 2 — 보습제 강도 추천기

- **URL**: `/tools/moisturizer-finder/`
- **입력**: 월령 · 피부 상태(정상/건조/거침/발진있음) · 계절 · 아토피 가족력 여부
- **출력**: 로션/크림/연고 강도 판정 + 성분 기준 카드 (세라마이드 ppm·pH·무향 여부) + 하루 도포 횟수 + 러베 제품 예시 1개
- **로직**: 아토피-보습제 글의 강도 매트릭스(월령×상태→강도) 도구화

### 도구 3 — 전성분 안전 체커

- **URL**: `/tools/ingredient-checker/`
- **입력**: 전성분 텍스트 붙여넣기 (또는 성분명 검색)
- **출력**: 아기 기준 주의 성분 하이라이트 (향료·알코올·정유·설페이트 등 카테고리별) + 등급 요약 + 성분별 한 줄 설명
- **데이터**: 주의 성분 사전 JSON (~200개 성분, EWG·SCCS·식약처 근거) — 자체 구축, 맘가이드 데이터 복제 금지
- **주의**: 특정 타사 제품 비방 금지 (표시광고법) — 성분 단위 정보만 제공

### 도구 4 — 목욕·환경 루틴 생성기

- **URL**: `/tools/bath-routine-builder/`
- **입력**: 월령 · 계절 · 피부 고민(태열/건조/없음)
- **출력**: 맞춤 루틴 카드 — 목욕 온도·시간·빈도, 실내 온습도, 보습 타이밍(soak & seal), 옷차림 한 겹 원칙 — 이미지 카드로 저장/공유 가능
- **로직**: 시기별 케어 글들의 표를 파라미터화

### 도구 5 — 자외선차단제 선택기

- **URL**: `/tools/sunscreen-picker/`
- **입력**: 월령 · 외출 시간대·길이 · 물놀이 여부 · 피부 민감도
- **출력**: 무기자차/유기자차 판정 + SPF/PA 기준 + 도포량·재도포 가이드 + 6개월 미만 = 차단제 대신 그늘·의류 권고 (AAP 기준)

### 초기 지원 글 25편 (도구별 5편 — 모두 신규 작성, ruuve-guide 복제 금지)

- **진단기 지원**: 판정 결과별 해설 — 태열이라면 / 땀띠라면 / 아토피 의심이라면 / 침독이라면 / 진료가 필요한 신호 총정리
- **보습 지원**: 세라마이드 함량 읽는 법 / 로션·크림·연고 차이 / 월령별 보습 강도 / soak & seal 실전 / 보습제 교체 시점
- **성분 지원**: 아기 화장품 주의 성분 카테고리 / 향료 표기 읽는 법 / EWG 등급의 의미와 한계 / 전성분 표 읽는 순서 / "무첨가" 마케팅 해석법
- **목욕 지원**: 월령별 목욕 온도·시간 / 계절별 실내 환경 / 목욕 후 3분 보습 / 신생아 첫 목욕 / 목욕 거부 대처
- **자외선 지원**: 무기자차 vs 유기자차 / 6개월 미만 자외선 원칙 / SPF·PA 읽는 법 / 재도포 실전 / 물놀이 차단제

각 글 2,500-4,000자, 도구 결과 페이지에서 근거 링크로 연결. **발행 페이스: 월 10편 상한.**

## 5. 디자인 방향 — "랩" 룩 (ruuve-guide 재사용 금지)

- **무드**: 클리니컬 + 친근 — 흰 배경, 데이터 카드, 게이지·차트 요소. ruuve-guide의 warm taupe/cream 매거진 룩과 정반대
- **컬러**: 프라이머리 `#0EA5A4`(teal — 신뢰·클리닉), 판정 신호등 `#22C55E`/`#F59E0B`/`#EF4444`, 배경 `#FFFFFF`/`#F8FAFC`
- **타이포**: Pretendard (한국 표준) — 단 크기·굵기 체계는 독자 설계
- **시그니처 요소**: 판정 게이지, 스텝 탭 인터페이스, 결과 카드(공유 가능 이미지), 신호등 뱃지
- **모바일 퍼스트**: 도구는 한 손 탭 조작 기준으로 설계

## 7. 신규 사이트 헌법 (페널티 재발 방지 — 영구 룰)

1. **발행 페이스 상한**: 월 10편 → 20편 → 40편. cron 자동 대량 발행 금지. 도구는 월 1-2개.
2. **도구 우선**: 새 자산 추가 시 "글 10편보다 도구 1개" 우선순위.
3. **ruuve-guide 콘텐츠 복제·번역·요약 재발행 절대 금지**. 로직·지식은 참고하되 텍스트는 전부 신규.
4. **상호 링크 최소화**: ruuve-guide ↔ babyskinlab 링크는 About의 운영사 소개 1곳만.
5. **주간 GSC 모니터링**: 매주 색인·노출 확인, 이상 신호 시 발행 중단 후 진단.
6. **YMYL 안전장치**: 모든 판정 결과에 "참고용, 진단 아님" 면책 + 진료 신호 체크리스트 필수.
7. **제품 노출 절제**: 도구 결과당 러베 제품 예시 최대 1개, "예시" 표기, 의약품적 효능 표방 금지(화장품법).
8. **misc**: slug 영문 (글로벌 여지), 날짜 실제 일자, 이미지 alt·dimensions 처음부터.
